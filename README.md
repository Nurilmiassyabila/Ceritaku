# Ceritaku
Ceritaku is a personal story-sharing web application built with modern JavaScript and Webpack. This project implements client-side routing, modular code structure, and API integration. It focuses on creating an intuitive interface for users to write, view, and manage their stories while exploring best practices in web development.
